
# Overview :
A Task Management System built using React, Node.js, Express, and MySQL. The application provides role-based authentication for Admin and Employees, task assignment, task tracking, profile management, and dashboard reports.

Tech Stack :
Frontend : ReactJS
Backend : NodeJS , ExpressJS , Mysql, JWT Aunthentication.

# To install the packages of both frontend and backend , connect to the same folder and install respeective pacakges.
ex.,  C:\Users\Gayatri\OneDrive\Documents\task-management-system_ass\frontend>
      C:\Users\Gayatri\OneDrive\Documents\task-management-system_ass\backend>

# Frontend Pckages :
npm install react-router-dom axios
npm install @mui/material @emotion/react @emotion/styled
npm install @mui/icons-material
npm install @mui/x-date-pickers dayjs
npm install react-toastify
npm install @mui/x-data-grid
npm install react-loader-spinner

# Backend Packages :
npm install express mysql2 cors multer jsonwebtoken bcrypt dotenv

# Installation : 

Clone the repository : git clone (repository_url)

# Frontend :
cd frontend 
npm i 
npm run dev

# Backend :
cd backend
npm i 
npm run dev
